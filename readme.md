# Ruby Sync Racing
An attempt at replicating the gameplay of Async Racing.

## Usage - Game
`bundle`

`ruby ruby_sync_racing.rb`

Options: `--editor` opens track editor.

## Usage - Server
`bundle`

`ruby server.rb`
